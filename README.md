# EduPlatform — Интерактивная платформа для онлайн-обучения

Дипломная работа. Тема: «Разработка интерактивной платформы для подбора курсов и онлайн-обучения».

## Быстрый старт

### Двойной клик по файлу:
```
start.bat
```
Скрипт сам установит зависимости и запустит оба сервера, затем откроет браузер.

### Или вручную:

**1. Backend (API):**
```bash
cd backend
npm install
node src/index.js
# Сервер на http://localhost:5000
```

**2. Frontend (React):**
```bash
cd frontend
npm install
npm start
# Приложение на http://localhost:3000
```

## Функциональность

- Главная страница с каталогом и поиском
- Фильтрация курсов по категории, уровню, цене
- Детальная страница курса с программой и отзывами
- Регистрация и авторизация (JWT)
- Личный кабинет с прогрессом обучения
- Страница просмотра уроков с отметкой прогресса
- Система отзывов и рейтингов

## Стек технологий

**Frontend:** React 18, React Router v6, Axios, CSS3
**Backend:** Node.js, Express, JWT, bcryptjs
**База данных:** In-memory (без внешней БД, всё в оперативной памяти)
